Opera Cloud

Setup & Login

  • Bookmark Prod Link: copy URL to top bar → drag to bookmarks → rename.
  • Log In via OKTA: use firstName.lastName@rwlasvegas.com + password → click OKTA in Opera.

Quoting & Booking

  • Quote Rates: explain nightly rate, resort fees, 48-hr cancellation, $150/night incidental hold, room type → add formatted note.
  • Booking: Look To Book → dates & pax → search → select tower/view → quote → select → create or find profile → enter CC → book → collect deposit → add notes.

Guest Profiles

  • Search Profile: try multiple search fields if no match.
  • Create Profile: click Create → fill info → save thrice if needed.

Deposits & Terminals

  • Deposit Codes: WV, CP, DD, CASH usage.
  • Post Deposit: Reservation → Deposit/Cancellation → ⋮ → Post Deposit → reference “1st Night Deposit”.
  • Select Terminal: GP tab → choose “DESK ##”.

Notes, Alerts & Traces

  • Notes: Reservation → Notes → New/Edit → title “gen” → save.
  • Alerts: Reservation → Alerts → New/Edit → titles MISC/B2B/PRE → save.
  • Traces: Reservation → Traces → New/Edit → titles FDK/HSK/GPD/ETC at 5 AM → save.

Preferences & Letters

  • Preferences: Reservation → Preferences → search “12PM” or “B2B” → add → save.
  • Confirmation Letters: Reservation → Confirmation Letters → send email → choose tower/template.

Change Log & Policies

  • Changes Log: I Want To → Changes Log → review → load more.
  • Cancellation & Refund: I Want To → Cancel → reason → Post Penalty (if needed) → reverse deposit → email receipt.

Modify Reservation

  • Change Rates/Room: Daily Details → adjust charges or room type → refresh.
  • Change Dates: Edit reservation dates → note original rates.

Additional Operations

  • Accompany Guest: Accompany Guest/Profile → New → add profile → Notes “SHARE”.
  • B2B: add Alert “B2B” + Note + Trace + Preference for back-to-back stays.
  • Room Assign/Move: I Want To → Assign Room → choose room → check Departures for B2B.
  • Pre-Register: set arrival time → if late, add Note/Alert/Preference or Trace for future.

Billing & Folios

  • Billing Windows: I Want To → Billing → Show More → know windows 1–8 & 101–108.
  • Post Charge: Billing → Post Charge → description & amount → post.
  • Adjust vs. Comp: adjust mistaken charges vs. comp food/fees.
  • Transfer Posting: select minibar charges → Transfer to room → refund via Post Payment.
  • Minibar & AR Disputes: move charges to Window 5 → notify email → add notes.
  • Routing: Routing Instructions → New/Edit comps → authorizer/profile → amount → save.
  • CC Authorizations: I Want To → CC History → view holds vs. charges → sort.
  • Process Check-Out: I Want To → Check Out or Billing → settle each window → send folio/receipt.
  • Send Folio Copy: I Want To → Billing → ⋮ → Informational Folio → Send by Email.
  • Close Folio: I Want To → Billing → I Want To → Close Folio.

Packages, Promos & Stay Details

  • Packages/Promos: Stay Details → Rate Code → Packages → Daily View & Breakout → comp unused credits.
  • Stay Details: view totals, rate codes, third-party info.

Service Orders

  • Create Order: Service Orders → + → room # → select service → remarks → submit.
  • Defer Order: select item → clock icon → set time → submit.
  • Cancel Order: select order → Void or call Housekeeping.
  • Check Status/History: Service Orders → Order # → Location → filter by date.
