Opera Cloud

Bookmarked Opera Cloud Prod Link Situational

  • Copy link from the browser's address bar
  • Drag it to your bookmarks bar
  • Right-click to edit the name

Logging In to Opera Cloud Prod

  • Use your OKTA credentials
  • Username format: firstName.lastName@rwlasvegas.com
  • Click OKTA at the bottom of Opera Cloud login page

Quoting Rates Important

When booking a room, inform guests about these 5 things:

  • Rate for each night
  • Resort fees (mention separately or include in total)
  • 48-hour cancellation policy
  • Incidental hold of $150/night (except suites)
  • Room type booked

After quoting, add a note to the reservation. Example: "K1PO Quoted: $79, $79 RF $55, RF $55 | Inc$150/night 48 CXL"

Booking a Reservation Important

  • Go to Look To Book page
  • Enter dates, number of people, and search
  • Ask guest about tower and view preferences
  • Click on rate, scroll down to see rates to quote
  • After quoting, click Select
  • Search for guest profile or create new
  • Collect credit card number and expiration
  • Click Book Now, collect deposit, add reference "1st Night Deposit"
  • Go to reservation and add notes

Profile Management

  • Use multiple search methods to find guest profiles
  • Create new profiles when needed - enter all information and save multiple times

Billing & Payments

Deposit Codes Important

  • WV: Waive Deposit (comps/B4 Checking)
  • CP: Chip & Pin (physical card placeholder)
  • DD: Standard payment option
  • CASH: Placeholder or for comp rooms

Collecting Deposits

  • Go to reservation → Deposit/Cancellation
  • Click ellipses (⋮) next to deposit amount
  • Select "Post Deposit"
  • Add reference "1st Night Deposit"
  • Process and email receipt if successful

Billing Windows Very Important

  • Window 1: Guest payment responsibility
  • Window 2: Third party/company payments
  • Window 5: Guest disputes
  • Windows 101-108: Comp windows
  • Enable post stay charging for checked out guests

Credit Card Authorizations Most Important

  • Go to Payment Instructions → Credit Card History
  • Shows authorizations, payments, refunds, failures
  • Authorization holds money, settlement charges card
  • Processing times: 24-48hrs (credit), 3-10 days (debit)

Reservation Management

Notes & Alerts Important

  • Notes: Detailed descriptions/full stories
  • Alerts: Quick notes for next person (MISC, B2B, Pre)
  • Traces: Future actions (FDK, HSK, GPD)

Cancellations & Refunds Important

  • Verify cancellable rate codes (DRACK, DBAR, YYGEN1)
  • I Want To → Cancel Reservation → Select reason
  • Post penalty if within 48 hours
  • Refund via Deposit/Cancellation → Reverse payment

Modifying Reservations

  • Rates/Room Types: Use Daily Details
  • Dates: Use Edit option on reservation
  • Never modify third-party (LL codes) without permission

B2B (Back-to-Back) Situational

  • Add alerts titled "B2B" with confirmation numbers
  • Add notes describing the situation
  • Set up FDK trace for 5 AM on second reservation
  • Add B2B preference

Guest Services

Luggage Assistance Situational

  • Call Front Services at Ext: 7696
  • Provide guest ticket number, last name, and room number
  • For checked out guests: room number, last name, bag count

Pre-Register for Late Arrivals Situational

  • Set arrival time using Edit reservation
  • For arrivals after 12 AM: assign room, create note, add "PRE" alert
  • For arrivals before 11 PM: assign room, create note, add "Late Arrival" preference

Service Orders Very Important

  • Create via Service Orders → Yellow plus
  • Enter room number and select service type
  • Add remarks if needed
  • Defer orders using clock icon for future times
  • Cancel orders by voiding (may require HSKP manager)

Minibar & Disputes Important

  • Under $100: Transfer charges to Minibar room (9046)
  • Over $100: Move charges to Window 5, email FDCashier and MinibarDisputes

Communication Systems

Cisco Finesse Situational

  • Bookmark URL by dragging to bookmarks bar
  • Statuses: NR (Not Ready), Break, Lunch
  • Check queues via Team tab
  • Transfer calls: Place on hold → Contact recipient → Transfer or Conference

Red Chat Important

  • Access via Cisco Finesse → Red Chat
  • Set status to Ready in agent settings
  • Save common responses in "My Responses"
  • Respond to pending chats by clicking "Pick Up"

Information Inbox

  • Access via different browser or incognito mode
  • Login: information@rwlv.onmicrosoft.com
  • Set email category to your name and move to completed when done

Other Systems

B4 Check In Important

  • Send invoices via Transform Mailer
  • Select reservation payment → single/multiple
  • Enter confirmation number and amount
  • Use reservation payment template

Grub Hub OTF

  • Troubleshooting: Ensure room number without dashes
  • Verify name spelling matches reservation
  • Select OTF option in Grub Hub
  • In-room dining available for Conrad Suites and all Crockfords

Nevotek Situational

  • Set wake-up calls via Guest Database
  • Delete wake-up calls in same interface
  • Failed wake-up: Call room multiple times, then security for wellness check

Synkros

  • Search patrons: GR Contact → Find tab
  • Create GR accounts: GR Contact → Enter details → Save
  • Search vouchers: Select profile → Offers tab
  • For room offers: Transfer to Resort Marketing (6106)
